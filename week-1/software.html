WEB DESIGING

Firstly, we were introduced to the internet and how it works. The difference between the web and the internet is that the web is just for connecting the internet to distant devices. The internet can still be in place when the web is not, while the web can't exist without the internet. We also learned about HTML and HTTP is for transporting data and HTML is for creating webpages. Additionally, we learned about the URL, which is the uniform resource locator that identifies the location of a specific resource on the internet. We also covered the IP, which is the internet protocol address that contains device location information and enables data sharing. We learned how to build software, which involves a group of people such as designers, business analysts, testing individuals, and back-end and front-end teams, etc.

We have implemented version control to keep track of changes and updates for the project. We have also integrated Visual Studio Code and Git to manage our files. In the "workspace" folder, we have created a sub-folder named "intro-to-git" and opened it in Visual Studio Code. Additionally, we have set up Git within Visual Studio Code using the terminal. First, we checked if Git was installed by typing "git --version". Then, we configured Git with our name and email using the commands "git config --global user.name your_name" and "git config --global user.email your_email@gmail.com". After that, we initialized Git in the project by running "git init". To review the status of changes, we used the command "git status". To stage all changes, we used "git add .", and lastly, we committed our changes to Git using the command "git commit -m "message””.
